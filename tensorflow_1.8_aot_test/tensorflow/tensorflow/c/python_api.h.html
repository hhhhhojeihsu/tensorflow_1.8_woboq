42// Sets whether ops missing a shape inference function should trigger an

46// Extends `session` with any new operations added to its associated graph.
47// Usually this happens automatically in TF_SessionRun. After this is called,
48// TF_SessionRun will no longer extend the session on every call.
49//
50// We expose this here to allow fine-grained synchronization in multi-threaded
51// workloads, which is required since the Python implementation depends on the
52// above mutation methods. This allows us to prevent modifications to nodes in
53// the graph after the session has been made aware of them.
54void ExtendSession(TF_Session* session, TF_Status* status);
55
56// Returns the serialized CppShapeInferenceResult::HandleData proto for
57// `output` if its a resource tensor, or otherwise returns the empty string.
58// TODO(b/74620627): remove when _USE_C_SHAPES is removed
59std::string ResourceHandleShapeAndType(TF_Graph* graph, TF_Output output);
