23// EXPERIMENTAL: tfdbg debugger-inserted ops.
24// These ops are used only internally by tfdbg. There is no API for users to
25// direct create them. Users can create them indirectly by using
26// RunOptions.debug_options during Session::Run() call. See tfdbg documentation
27// for more details.
28REGISTER_OP("Copy")
29 .Input("input: T")
30 .Output("output: T")
31 .Attr("T: type")
32 .Attr("tensor_name: string = ''")
33 .Attr("debug_ops_spec: list(string) = []")
34 .SetAllowsUninitializedInput()
35 .SetShapeFn(shape_inference::UnchangedShape)
36 .Doc(R"doc(
37Copy Op.
38
39Performs CPU-to-CPU or GPU-to-GPU deep-copying of tensor, depending on the
40device on which the tensor is allocated.
41N.B.: If the all downstream attached debug ops are disabled given the current
42gRPC gating status, the output will simply forward the input tensor without
43deep-copying. See the documentation of Debug* ops for more details.
44
45Unlike the CopyHost Op, this op does not have HostMemory constraint on its
46input or output.
47
48input: Input tensor.
49output: Output tensor, deep-copied from input.
50tensor_name: The name of the input tensor.
51debug_ops_spec: A list of debug op spec (op, url, gated_grpc) for attached debug
52 ops. Each element of the list has the format
53 <debug_op>;<grpc_url>;<gated_grpc>, wherein gated_grpc is boolean represented
54 as 0/1. E.g., "DebugIdentity;grpc://foo:3333;1",
55 "DebugIdentity;file:///tmp/tfdbg_1;0".
56)doc");

60 .Output("output: T")
61 .Attr("T: type")
62 .Attr("tensor_name: string = ''")
63 .Attr("debug_ops_spec: list(string) = []")
64 .SetAllowsUninitializedInput()
65 .SetShapeFn(shape_inference::UnchangedShape)
66 .Doc(R"doc(
67Copy Host Op.
68
69Performs CPU-to-CPU deep-copying of tensor.
70N.B.: If the all downstream attached debug ops are disabled given the current
71gRPC gating status, the output will simply forward the input tensor without

74Unlike the Copy Op, this op has HostMemory constraint on its input or output.

76input: Input tensor.
77output: Output tensor, deep-copied from input.
78tensor_name: The name of the input tensor.
79debug_ops_spec: A list of debug op spec (op, url, gated_grpc) for attached debug
80 ops. Each element of the list has the format
81 <debug_op>;<grpc_url>;<gated_grpc>, wherein gated_grpc is boolean represented
82 as 0/1. E.g., "DebugIdentity;grpc://foo:3333;1",
83 "DebugIdentity;file:///tmp/tfdbg_1;0".
84)doc");

86REGISTER_OP("DebugIdentity")
87 .Input("input: T")
88 .Output("output: T")
89 .Attr("T: type")
90 .Attr("device_name: string = ''")
91 .Attr("tensor_name: string = ''")
92 .Attr("debug_urls: list(string) = []")
93 .Attr("gated_grpc: bool = false")
94 .SetAllowsUninitializedInput()
95 .SetShapeFn(shape_inference::UnchangedShape)
96 .Doc(R"doc(
97Debug Identity Op.
98
99Provides an identity mapping of the non-Ref type input tensor for debugging.
100
101input: Input tensor, non-Reference type.
102output: Output tensor that equals the input tensor.
103tensor_name: Name of the input tensor.
104debug_urls: List of URLs to debug targets, e.g.,
105 file:///foo/tfdbg_dump, grpc:://localhost:11011
106gated_grpc: Whether this op will be gated. If any of the debug_urls of this
107 debug node is of the grpc:// scheme, when the value of this attribute is set
108 to True, the data will not actually be sent via the grpc stream unless this
109 debug op has been enabled at the debug_url. If all of the debug_urls of this
110 debug node are of the grpc:// scheme and the debug op is enabled at none of
111 them, the output will be an empty Tensor.
112)doc");

134gated_grpc: Whether this op will be gated. If any of the debug_urls of this

137 debug op has been enabled at the debug_url. If all of the debug_urls of this

140)doc");

142REGISTER_OP("DebugNumericSummary")
143 .Input("input: T")
144 .Output("output: double")
145 .Attr("T: type")
146 .Attr("device_name: string = ''")
147 .Attr("tensor_name: string = ''")
148 .Attr("debug_urls: list(string) = []")
149 .Attr("lower_bound: float = -inf")
150 .Attr("upper_bound: float = inf")
151 .Attr("mute_if_healthy: bool = false")
152 .Attr("gated_grpc: bool = false")
153 .SetAllowsUninitializedInput()
154 // Note: this could return a more specific shape if needed in future.
155 .SetShapeFn(shape_inference::UnknownShape)
156 .Doc(R"doc(
157Debug Numeric Summary Op.
158
159Provide a basic summary of numeric value types, range and distribution.
160
161input: Input tensor, non-Reference type, float or double.
162output: A double tensor of shape [14 + nDimensions], where nDimensions is the
163 the number of dimensions of the tensor's shape. The elements of output are:
164 [0]: is initialized (1.0) or not (0.0).
165 [1]: total number of elements
166 [2]: NaN element count
167 [3]: generalized -inf count: elements <= lower_bound. lower_bound is -inf by
168 default.
169 [4]: negative element count (excluding -inf), if lower_bound is the default
170 -inf. Otherwise, this is the count of elements > lower_bound and < 0.
171 [5]: zero element count
172 [6]: positive element count (excluding +inf), if upper_bound is the default
173 -inf. Otherwise, this is the count of elements < upper_bound and > 0.
174 [7]: generalized +inf count, elements >= upper_bound. upper_bound is +inf by
175 default.
176Output elements [1:8] are all zero, if the tensor is uninitialized.
177 [8]: minimum of all non-inf and non-NaN elements.
178 If uninitialized or no such element exists: +inf.
179 [9]: maximum of all non-inf and non-NaN elements.
180 If uninitialized or no such element exists: -inf.
181 [10]: mean of all non-inf and non-NaN elements.
182 If uninitialized or no such element exists: NaN.
183 [11]: variance of all non-inf and non-NaN elements.
184 If uninitialized or no such element exists: NaN.
185 [12]: Data type of the tensor encoded as an enum integer. See the DataType
186 proto for more details.
187 [13]: Number of dimensions of the tensor (ndims).
188 [14+]: Sizes of the dimensions.
189
190tensor_name: Name of the input tensor.
191debug_urls: List of URLs to debug targets, e.g.,
192 file:///foo/tfdbg_dump, grpc:://localhost:11011
193lower_bound: (float) The lower bound <= which values will be included in the
194 generalized -inf count. Default: -inf.
195upper_bound: (float) The upper bound >= which values will be included in the
196 generalized +inf count. Default: +inf.
197mute_if_healthy: (bool) Do not send data to the debug URLs unless at least one
198 of elements [2], [3] and [7] (i.e., the nan count and the generalized -inf and
199 inf counts) is non-zero.
200gated_grpc: Whether this op will be gated. If any of the debug_urls of this
201 debug node is of the grpc:// scheme, when the value of this attribute is set
202 to True, the data will not actually be sent via the grpc stream unless this
203 debug op has been enabled at the debug_url. If all of the debug_urls of this
204 debug node are of the grpc:// scheme and the debug op is enabled at none of
205 them, the output will be an empty Tensor.
206
207)doc");
