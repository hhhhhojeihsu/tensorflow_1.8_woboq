25// Traverse the graph in reverse dfs order, starting from the list of nodes
26// specified in the 'from' argument. The pre_order and post_order functors will
27// be called on each reachable node (including the 'from' nodes) in pre and post
28// order. If loops are found, the on_back_edge functor will be called on the
29// corresponding back edges. Moreover, the pre and post order will assume that
30// these back edges will be cut.
31void ReverseDfs(const GraphView& graph_view, const std::vector<NodeDef*>& from,
32 const std::function<void(NodeDef*)>& pre_order,
33 const std::function<void(NodeDef*)>& post_order,
34 const std::function<void(NodeDef*, NodeDef*)>& on_back_edge);
35
