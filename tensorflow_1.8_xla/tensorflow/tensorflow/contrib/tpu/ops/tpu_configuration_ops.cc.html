26// Configuring a distributed TPU system is achieved by running
27// the following Ops:
28//
29// 1 Run _DisconnectHostFromDistributedTPUSystem on the TPU_SYSTEM of each
30// host. This is needed in case the system had previously been configured. It
31// returns, for each host, the number of TPU chips on the host.
32//
33// 2 Run _ConfigureDistributedTPU on TPU_SYSTEM of worker 0. Takes as input the
34// number of chips on each host. Validates that all hosts have the same number
35// of chips, and that the chips are consistent with the topology set by
36// flags. Has a single output which is a proto describing the requested system
37// configuration, which is sent to all hosts.
38//
39// 3 Run _InitializeHostForDistributedTPU on the TPU_SYSTEM of each host, taking
40// as input the output from ConfigureDistributedTPU. Has a single Tensor output
41// which is a vector of int32 indicating, for each TPU on the host, what its
42// global TPU system id is.
43//
44// 4 Run _WaitForDistributedTPU on TPU_SYSTEM, taking as input the
45// outputs from all the _InitializeHostForDistributedTPU
46// Ops. _These partial specs are combined in the Op with the outputs from
47// the host initialization Ops to construct a mapping from full TPU device
48// specs to global TPU ids. Has a single Tensor output which is a
49// matrix of int32 indicating, for each host (outer dimension) and for
50// each TPU on the host (inner dimension) what that TPU's global id
51// is. _WaitForDistributedTPU also waits for the TPU distributed
52// system to initialize fully, which may take several minutes for a
53// large system.
54//
55// 5 Run _SetGlobalTPUArray on the TPU_SYSTEM of each host, taking as input the
56// output from _WaitForDistributedTPU. This Op tells each host the global Id of
57// every TPU on every host.
58//
59// Most user code works by placing the ConfigureDistributedTPU Op on the desired
60// TPU_SYSTEM device, and a graph rewrite replaces it by the subgraph described
61// above.
62//
63//
64// A distributed TPU system can be cleanly shut down by running the following
65// Ops:
66//
67// 1 Run _DisconnectHostFromDistributedTPUSystem on the TPU_SYSTEM of each host.
68//
69// 2 Run _ShutdownDistributedTPU on the TPU_SYSTEM where
70// _ConfigureDistributedTPU was run. The Op will return an error if no system is
71// configured.
72//
73//
74// Most user code works by placing the ShutdownDistributedTPU Op on the desired
75// TPU_SYSTEM device, and a graph rewrite replaces it by the subgraph described
76// above.
77
78REGISTER_OP("_ConfigureDistributedTPU")
79 .Input("inputs: N * int32")
80 .Output("output: string")
81 .Attr("N: int >= 1")
82 .SetIsStateful()
83 .SetShapeFn([](InferenceContext* c) {
84 ShapeHandle input;
85 // Validate that all the inputs are scalars.
86 for (int i = 0; i < c->num_inputs(); ++i) {
87 TF_RETURN_IF_ERROR(c->WithRank(c->input(i), 0, &input));
88 }
89 c->set_output(0, c->Scalar());
90 return Status::OK();
91 })
92 .Doc(R"doc(
93An op that sets up the centralized structures for a distributed TPU
94system.
95
96inputs: A scalar tensor for each host indicating how many TPU chips
97there are on the host.
98output: A tensor containing a TPUHostConfiguration proto serialized to
99a string, containing the information necessary to initialize the chips
100in a host.
101)doc");
102
103REGISTER_OP("_WaitForDistributedTPU")
104 .Input("inputs: N * int32")
105 .Output("topology: string")
106 .Attr("startup_timeout_sec: int = 20")
107 .Attr("N: int")
108 .SetIsStateful()
109 .SetShapeFn([](InferenceContext* c) {
110 ShapeHandle input;
111 // Validate that all the inputs have the same vector shape.
112 for (int i = 0; i < c->num_inputs(); ++i) {
113 TF_RETURN_IF_ERROR(c->WithRank(c->input(i), 1, &input));
114 }
115 c->set_output(0, c->Scalar());
116 return ::tensorflow::Status::OK();
117 })
118 .Doc(R"doc(
119An op that blocks execution until a distributed TPU system has
120started up. This Op must be run on the same TPU_SYSTEM device as
121_ConfigureDistributedTPU, and takes an inputs the outputs from the
122_InitializeHostForDistributedTPU Ops.
123
124inputs: For each initialized host, a vector giving the global TPU id
125of each TPU on the host.
126topology: A serialized tensorflow.tpu.TopologyProto that describes the TPU
127topology.
128startup_timeout_sec: The number of seconds to wait for the TPU system
129to stabilize.
130)doc");
131
132REGISTER_OP("_SetGlobalTPUArray")
133 .Input("topology: string")
134 .SetIsStateful()
135 .SetShapeFn([](InferenceContext* c) {
136 ShapeHandle input;
137 TF_RETURN_IF_ERROR(c->WithRank(c->input(0), 0, &input));
138 return ::tensorflow::Status::OK();
139 })
140 .Doc(R"doc(
141An op that informs a host of the global ids of all the of TPUs in the
142system.
143
144topology: A serialized tensorflow.tpu.TopologyProto that describes the TPU
145topology.
146)doc");
147
148REGISTER_OP("_ShutdownDistributedTPU")
149 .SetIsStateful()
150 .SetShapeFn(shape_inference::UnknownShape)
151 .Doc(R"doc(
152An op that shuts down a running distributed TPU system. The Op returns
153an error if no system is running. This Op must be run on the same
154TPU_SYSTEM device as the corresponding _ConfigureDistributedTPU was run
155to start the system, and must be run only after
156_DisconnectHostFromDistributedTPUSystem has completed on every host in
157the system.
158)doc");
159
160REGISTER_OP("_InitializeHostForDistributedTPU")
161 .Input("input: string")
162 .Output("tpu_ids: int32")
163 .SetIsStateful()
164 .SetShapeFn([](InferenceContext* c) {
165 ShapeHandle input;
166 TF_RETURN_IF_ERROR(c->WithRank(c->input(0), 0, &input));
167 c->set_output(0, c->Vector(c->UnknownDim()));
168 return ::tensorflow::Status::OK();
169 })
170 .Doc(R"doc(
171An op that connects each chip on the host to a centralized UberDriver to allow
172them to operate as a distributed system with chips in other hosts.
173
174input: A string containing the address of the UberDriver to connect to.
175tpu_ids: A vector containing the global TPU id of each TPU on the host.
176)doc");
177
178REGISTER_OP("_DisconnectHostFromDistributedTPUSystem")
179 .Output("number_of_tpu_chips: int32")
180 .SetIsStateful()
181 .SetShapeFn(shape_inference::UnknownShape)
182 .Doc(R"doc(
183An op that disconnects the TPUs on a host from a running distributed
184TPU system.
185
186number_of_tpu_chips: A scalar tensor containing the number of TPU
187chips on the host.
188)doc");
189
190REGISTER_OP("ConfigureDistributedTPU")
191 .Output("topology: string")
192 .Attr("embedding_config: string = ''")
193 .Attr("tpu_embedding_config: string = ''")
194 .Attr("is_global_init: bool = false")
195 .SetIsStateful()
196 .SetShapeFn(shape_inference::UnknownShape)
197 .Doc(R"doc(
198An op that sets up the centralized structures for a distributed TPU
199system.
200
201topology: A serialized tensorflow.tpu.TopologyProto that describes the TPU
202topology.
203tpu_embedding_config: Serialized tensorflow.tpu.TPUEmbeddingConfiguration that
204describes the embedding lookups of the program.
205embedding_config: Reserved. Do not use.
206is_global_init: Reserved. Do not use.
207)doc");
208
209REGISTER_OP("ShutdownDistributedTPU")
210 .SetIsStateful()
211 .SetShapeFn(shape_inference::UnknownShape)
212 .Doc(R"doc(
213An op that shuts down a running distributed TPU system. The Op returns
214an error if no system is running.
215)doc");

217REGISTER_OP("SessionStatus")
218 .Input("fetch_start_timestamp: double")
219 .Output("status: string")
220 .SetShapeFn(shape_inference::ScalarShape)
221 .Doc(R"doc(
222Not for public usage.
223
224Returns messages from the current session as a serialized SessionStatusProto.
225
226This includes the current state of the compiler, along with any critical
227logging or warning messages.
228
229fetch_start_timestamp: any messages earlier than this will be excluded from the
230returned proto.
231)doc");
