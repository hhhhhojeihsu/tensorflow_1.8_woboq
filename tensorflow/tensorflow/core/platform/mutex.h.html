44// Like "cv->wait(*mu)", except that it only waits for up to "ms" milliseconds.
45//
46// Returns kCond_Timeout if the timeout expired without this
47// thread noticing a signal on the condition variable. Otherwise may
48// return either kCond_Timeout or kCond_MaybeNotified
